## ❗️ Troubleshooting

| Problem               | Solution                                       |
|-----------------------|------------------------------------------------|
| Application crash     | Update to the latest certified GPU drivers     |
| Performance lag       | Enable Hardware Acceleration in System tab     |
| Missing toolbars      | Reset workspace to "Drafting & Annotation"     |
| Security alert        | Add the installation path to safe exclusions    |
| DWG incompatibility   | Use the included legacy conversion utility      |
